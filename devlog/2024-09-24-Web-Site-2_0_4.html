Web Site v2.0.4 launch - Written Devlog

While it may appear a bit strange to give a seemingly "minor update" its own written devlog, this particular update is quite larger and more expansive than the others from this month.

This release has a good amount of moving parts to it, so here's a semi-detailed list on what's changed around the site.
-----
- the Blog Backlog has been overhauled, now sorting posts by year
- these changes were made due to the ever-growing size of the Backlog, and how out-of-control it would eventually get
- almost every Games sub-page has been redesigned, providing more info on the respective games
- the previous game pages were either boring or wasting system resources (the embeds), so they've been changed
- a new website is now featured in the About page
- i've been a fan of this creator's work ever since 2022, and now his new website is featured on this site!
- minor changes to a variety of pages
- some changes include text edits/typo fixes, layout tweaks, and minor polish
- removed the Infinity Player+ and EggWars+ sub-pages
- the Remastered Collection sub-page entirely deprecates these pages' usefulness, so they've been scrubbed
- we reached 200 unique visitors on the site!
- this is just a fun one :)
-----
Hope you all enjoy this neat little update. Take care!
